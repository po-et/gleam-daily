# 贡献指南

感谢你对拾光日报的兴趣！

## 提 Issue

- **Bug**：请附 macOS 版本、应用版本、复现步骤，以及「设置 → AI 引擎」用的是哪种引擎
- **功能建议**：先说清场景和痛点，再谈方案；隐私相关的设计红线见 README「隐私设计」一节

## 提 PR

1. Fork 并从 `main` 拉分支
2. 改动前先读 `docs/SPEC.md`（产品与技术契约）与 `docs/DESIGN.md`（UI 设计系统）——
   **契约层（`src/shared`、preload、DB Schema）的改动请先开 Issue 讨论**
3. 保证 `npm run typecheck` 与 `npm run build` 通过
4. UI 改动请附截图（浅色 + 深色两套主题）
5. 提交信息用中文或英文皆可，说清「为什么」而不只是「做了什么」

## 隐私红线（不接受违反以下原则的 PR）

- 不引入任何遥测、统计上报、账号体系
- 用户数据不得离开本机（AI 请求直连用户配置的服务商除外）
- 截图必须在分析后删除，敏感熔断逻辑不得绕过
- API Key 只能加密存储、只在主进程解密

## 代码风格

- TypeScript 严格模式；注释与错误信息用中文，面向"下一个读代码的人"
- 渲染层禁止引入 UI 组件库 / CSS 框架 / 图表库，样式一律手写并复用 `theme.css` 令牌
- 遵循现有目录分层（shared / main / preload / renderer），新 IPC 通道同步更新 SPEC
